# WorkTick

桌面端时间追踪应用，基于任务的项目管理工具。

## 功能特性

- **项目管理** - 创建和管理项目，支持自定义颜色和描述
- **任务管理** - 任务优先级（高/中/低）、状态流转（待办/进行中/已完成/已搁置）
- **时间追踪** - 任务级计时，支持暂停/继续，停止时选择任务状态
- **工作记录** - 为任务添加里程碑式的工作内容记录
- **仪表盘** - 今日工时、完成任务、活跃项目等数据概览
- **数据统计** - 每日/项目维度的工时统计

## 技术栈

- **前端**: Vue 3 + TypeScript + Pinia
- **桌面端**: Electron
- **数据库**: sql.js (SQLite)
- **构建**: Vite + electron-builder

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务
npm run dev

# 构建应用
npm run build
```

## 系统要求

- Node.js >= 18
- npm >= 9
